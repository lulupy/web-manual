# CSS3
css3不是一门新的语言, 它只是css的最新版本, 3是它的版本号,它在css2基础上增加了很多强大的功能.

目前主流浏览器chrome, safari, firfox, opera, 甚至是360都已经支持了css3的大部分功能. IE10以后也开始全面支持css3了.

###  一些新特性:
+  边框圆角
+  盒模型阴影和文字阴影
+  颜色, 不透明和渐变
+  web字体
+  2d/3d转换
+  过渡
